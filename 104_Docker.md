# 104. Docker

## Цель работы

- Познакомиться с **Docker**.
- Изучить некоторые базовые команды **Linux**.

## Задача

1. Создать и запустить контейнер на базе минимального образа (например, **ubuntu** или **alpine**).
2. Подключиться к контейнеру с помощью команды:

   ```bash
   docker exec -it CONTAINER_NAME sh
   ```

   Все последующие действия осуществляются внутри контейнера.

3. Установить **bash** (если он еще не установлен). Все дальнейшие действия производятся в **bash**. 4. В корне домашнего каталога (`~/`) создать файл `ФамилияИО.txt`(ваша фамилия и инициалы в имени файла).

4. Использовать команду `date`, чтобы вывести текущий день недели.
5. Вывести расширенную информацию о файле `/bin/bash`. Ответьте в очете на вопрос: что из себя представляет файл `/bin/bash`.

6. Скачать текст "Lewis Carroll. Alice's Adventures in Wonderland" с помощью `wget` или `curl` по следующему адресу: <http://www.lib.ru/CARROLL/alice.txt>.

7. Переименовать скачанный файл в `alice.txt`.
8. Используя команду `wc`, вывести только количество слов в `alice.txt`.
9. Показать первые 19 строк из `alice.txt`.
10. Показать последние 17 строк из `alice.txt`.
11. Создать каталоги `test1` ... `test40` с помощью команды `mkdir`.
12. Удалить только каталоги с именами, заканчивающимися на `3` и `7`, с помощью команд `rm`, `rmdir`.
13. Вывести список каталогов и перенаправить его в текстовый файл `test.txt`.
14. Вывести количество строк в `test.txt`.
15. В каталогах, номера которых кратны 5, создать текстовый файл `date.txt` с текущей датой в формате: `ГОД-МЕСЯЦ-ДЕНЬ`.
16. В каталогах, номера которых кратны 10, дописать в текстовый файл `date.txt` текущее время.
