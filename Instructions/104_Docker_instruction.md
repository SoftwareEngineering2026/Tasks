# Инструкция по выполнению лабораторной работы 105. Docker

## Шаг 1: Установите Docker

Перед началом работы убедитесь, что Docker установлен на вашем компьютере. Проверьте установку командой:

```bash
docker --version
```

Если Docker не установлен, следуйте [официциальной инструкции по установке Docker](https://docs.docker.com/get-docker/).

---

## Шаг 2: Создайте и запустите контейнер

Создайте контейнер на основе минимального образа (например, ubuntu):

```bash
docker run -it --name my_container ubuntu:latest
```

- `docker run`: создает и запускает контейнер.
- `-it`: позволяет взаимодействовать с контейнером через терминал.
- `--name my_container`: задает имя контейнера.
- `ubuntu:latest`: указывает, что мы используем образ Ubuntu последней версии.

---

## Шаг 3: Подключитесь к контейнеру

После запуска контейнера выполните команду для доступа к нему:

```bash
docker exec -it my_container sh
```

- `docker exec`: позволяет выполнять команды в уже запущенном контейнере.
- `-it`: также позволяет использовать интерактивный режим.
- `my_container`: имя вашего контейнера.

---

## Шаг 4: Установите bash

Внутри контейнера установите bash:

```bash
apt-get update && apt-get install -y bash
```

- `apt-get update`: обновляет список доступных пакетов.
- `apt-get install -y bash`: устанавливает пакет bash без запроса подтверждения.

---

## Шаг 5: Создайте файл ФамилияИО.txt

Создайте текстовый файл с вашим именем:

```bash
touch ~/ФамилияИО.txt
```

- `touch`: создает новый файл или обновляет время последнего доступа к существующему.

---

## Шаг 6: Выведите текущий день недели

Используйте команду `date`:

```bash
date +"%A"
```

- `date`: команда для отображения текущей даты и времени.
- `"%A"`: спецификатор формата для отображения полного названия дня недели.

---

## Шаг 7: Выведите информацию о файле /bin/bash

Используйте команду `ls` с опцией `-l`:

```bash
ls -l /bin/bash
```

- `ls -l`: выводит подробную информацию о файле, включая права доступа, владельца и размер.

---

## Шаг 8: Скачайте текст Alice's Adventures in Wonderland

С помощью `wget` или `curl` загрузите файл:

```bash
wget http://www.lib.ru/CARROLL/alice.txt
```

или

```bash
curl -O http://www.lib.ru/CARROLL/alice.txt
```

- `wget`: утилита для скачивания файлов из сети.
- `curl -O`: скачивает файл с сохранением оригинального имени.

---

## Шаг 9: Переименуйте скачанный файл

Переименуйте его в `alice.txt`:

```bash
mv alice.txt alice.txt
```

- `mv`: команда для переименования или перемещения файлов.

---

## Шаг 10: Подсчет слов в `alice.txt`

Используйте команду `wc`:

```bash
wc -w alice.txt
```

- `wc -w`: считает количество слов в файле.

---

## Шаг 11: Показать первые 19 строк

Для этого используйте команду `head`:

```bash
head -n 19 alice.txt
```

- `head -n`: выводит первые n строк файла.

---

## Шаг 12: Показать последние 17 строк

Используйте команду `tail`:

```bash
tail -n 17 alice.txt
```

- `tail -n`: выводит последние n строк файла.

---

## Шаг 13: Создание каталогов

Создайте каталоги `test1` ... `test40`:

```bash
for i in {1..40}; do mkdir "test$i"; done
```

- `for i in {1..40}`: итерация от 1 до 40.
- `mkdir`: создает директории.

---

## Шаг 14: Удаление каталогов

Удалите только каталоги, заканчивающиеся на `3` и `7`:

```bash
rm -r test{3,7}
```

- `rm -r`: рекурсивно удаляет каталоги.

---

## Шаг 15: Вывод списка каталогов

Выведите список оставшихся каталогов и перенаправьте его в `test.txt`:

```bash
ls -d test*/ > test.txt
```

- `ls -d`: выводит директории в формате списка.
- `>`: перенаправляет вывод в файл.

---

## Шаг 16: Подсчет строк в test.txt

Посчитайте количество строк в `test.txt`:

```bash
wc -l test.txt
```

- `wc -l`: считает количество строк в файле.

---

## Шаг 17: Создание файла date.txt

Создайте текстовый файл с текущей датой в каталогах, кратных 5:

```bash
for i in {1..40}; do
  if [ $((i % 5)) -eq 0 ]; then
    echo $(date +"%Y-%m-%d") > "test$i/date.txt"
  fi
done
```

---

## Шаг 18: Дописывание времени

В каталогах, кратных 10, допишите текущее время в `date.txt`:

```bash
for i in {1..40}; do
  if [ $((i % 10)) -eq 0 ]; then
    echo $(date +"%H:%M:%S") >> "test$i/date.txt"
  fi
done
```

- `>>`: дописывает данные в файл.

## Шаг 19: Создание Dockerfile

На этом шаге вам нужно будет записать все команды из предыдущих шагов в .sh скрипт и создать **Dockerfile**, который создает образ и выполняет этот скрипт в контейнере

```Dockerfile
    # Используем минимальный образ Ubuntu
    FROM ubuntu:latest

    # Устанавливаем рабочую директорию
    WORKDIR /home

    # Устанавливаем необходимые пакеты
    RUN apt-get update && apt-get install -y wget bash

    # Копируем скрипт в контейнер
    COPY script.sh /usr/local/bin/script.sh

    # Делаем скрипт исполняемым
    RUN chmod +x /usr/local/bin/script.sh

    # Выполняем скрипт при запуске контейнера
    CMD ["/bin/bash", "/usr/local/bin/script.sh"]
```
