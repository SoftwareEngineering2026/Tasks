# Инструкция по выполнению лабораторной работы 105. Docker

## Шаг 1: Установите Docker

Перед началом работы убедитесь, что Docker установлен на вашем компьютере. Проверьте установку командой:

```bash
docker --version
```

Если Docker не установлен, следуйте [официальной инструкции по установке Docker](https://docs.docker.com/get-docker/).

---

## Шаг 2: Создайте и запустите контейнер

Создайте контейнер на основе минимального образа (например, ubuntu):

```bash
docker run -it --name my_container ubuntu:latest
```

- `docker run`: создает и запускает контейнер.
- `-it`: позволяет взаимодействовать с контейнером через терминал.
- `--name my_container`: задает имя контейнера.
- `ubuntu:latest`: указывает, что мы используем образ Ubuntu последней версии.

---

## Шаг 3: Установите необходимые пакеты

Внутри контейнера установите необходимые пакеты:

```bash
apt-get update && apt-get install -y bash wget curl
```

- `apt-get update`: обновляет список доступных пакетов.
- `apt-get install -y`: устанавливает пакеты без запроса подтверждения.
- `bash`, `wget`, `curl`: необходимые утилиты для выполнения последующих шагов.

---

## Шаг 4: Создайте файл ФамилияИО.txt

Создайте текстовый файл с вашим именем:

```bash
cd ~
touch ~/ФамилияИО.txt
```

- `touch`: создает новый файл или обновляет время последнего доступа к существующему.

---

## Шаг 5: Выведите текущий день недели

Используйте команду `date`:

```bash
date +"%A"
```

- `date`: команда для отображения текущей даты и времени.
- `"%A"`: спецификатор формата для отображения полного названия дня недели.

---

## Шаг 6: Выведите информацию о файле /bin/bash

Используйте команду `ls` с опцией `-l`:

```bash
ls -l /bin/bash
```

- `ls -l`: выводит подробную информацию о файле, включая права доступа, владельца и размер.

---

## Шаг 7: Скачайте текст Alice's Adventures in Wonderland

С помощью `wget` или `curl` загрузите файл:

```bash
wget http://www.lib.ru/CARROLL/alice.txt
```

или

```bash
curl -O http://www.lib.ru/CARROLL/alice.txt
```

- `wget`: утилита для скачивания файлов из сети.
- `curl -O`: скачивает файл с сохранением оригинального имени.

---

## Шаг 8: Переименуйте скачанный файл

Переименуйте его в `alice_text.txt`:

```bash
mv alice.txt alice_text.txt
```

- `mv`: команда для переименования или перемещения файлов.

## Шаг 9: Подсчет слов в `alice_text.txt`

wc -w alice_text.txt

- `wc -w`: считает количество слов в файле.

## Шаг 10: Показать первые 19 строк

Для этого используйте команду `head`:

```bash
head -n 19 alice_text.txt
```

- `head -n`: выводит первые n строк файла.

---

## Шаг 11: Показать последние 17 строк

Используйте команду `tail`:

```bash
tail -n 17 alice_text.txt
```

- `tail -n`: выводит последние n строк файла.

---

## Шаг 12: Создание каталогов

Создайте каталоги `test1` ... `test40`:

```bash
for i in {1..40}; do mkdir "test$i"; done
```

- `for i in {1..40}`: итерация от 1 до 40.
- `mkdir`: создает директории.

---

## Шаг 13: Удаление каталогов

Удалите только каталоги, заканчивающиеся на `3` и `7`:

```bash
rm -r test3 test7 test13 test17 test23 test27 test33 test37
```

или используйте расширенный синтаксис:

```bash
for i in {1..40}; do
  if [ $((i % 10)) -eq 3 ] || [ $((i % 10)) -eq 7 ]; then
    rm -r "test$i"
  fi
done
```

- `rm -r`: рекурсивно удаляет каталоги.

**Примечание:** исходная команда `rm -r test{3,7}` удаляет только `test3` и `test7`, но не остальные каталоги, заканчивающиеся на 3 и 7 (test13, test17, test23, test27, test33, test37).

---

## Шаг 14: Вывод списка каталогов

Выведите список оставшихся каталогов и перенаправьте его в `test.txt`:

```bash
ls -d test*/ > test.txt
```

- `ls -d`: выводит директории в формате списка.
- `>`: перенаправляет вывод в файл.

---

## Шаг 15: Подсчет строк в test.txt

Посчитайте количество строк в `test.txt`:

```bash
wc -l test.txt
```

- `wc -l`: считает количество строк в файле.

---

## Шаг 16: Создание файла date.txt

Создайте текстовый файл с текущей датой в каталогах, кратных 5:

```bash
for i in {1..40}; do
  if [ $((i % 5)) -eq 0 ]; then
    echo $(date +"%Y-%m-%d") > "test$i/date.txt"
  fi
done
```

- `echo`: выводит текст.
- `>`: перенаправляет вывод в файл (создает или перезаписывает файл).

---

## Шаг 17: Дописывание времени

В каталогах, кратных 10, допишите текущее время в `date.txt`:

```bash
for i in {1..40}; do
  if [ $((i % 10)) -eq 0 ]; then
    echo $(date +"%H:%M:%S") >> "test$i/date.txt"
  fi
done
```

- `>>`: дописывает данные в файл (не перезаписывает).

---

## Шаг 18: Создание Dockerfile

На этом шаге вам нужно записать все команды из предыдущих шагов в скрипт `.sh` и создать **Dockerfile**, который создает образ и выполняет этот скрипт в контейнере.

### Пример файла `script.sh`:

```bash
#!/bin/bash

# Создание файла с именем
touch ~/ФамилияИО.txt

# Вывод дня недели
echo "День недели:"
date +"%A"

# Информация о файле bash
echo "Информация о /bin/bash:"
ls -l /bin/bash

# Скачивание текста
cd ~
wget -q http://www.lib.ru/CARROLL/alice.txt || curl -O http://www.lib.ru/CARROLL/alice.txt

# Переименование файла
mv alice.txt alice_text.txt

# Подсчет слов
echo "Количество слов в alice_text.txt:"
wc -w alice_text.txt

# Первые 19 строк
echo "Первые 19 строк:"
head -n 19 alice_text.txt

# Последние 17 строк
echo "Последние 17 строк:"
tail -n 17 alice_text.txt

# Создание каталогов
for i in {1..40}; do mkdir "test$i"; done

# Удаление каталогов, заканчивающихся на 3 и 7
for i in {1..40}; do
  if [ $((i % 10)) -eq 3 ] || [ $((i % 10)) -eq 7 ]; then
    rm -r "test$i"
  fi
done

# Вывод списка каталогов
ls -d test* > test.txt

# Подсчет строк
echo "Количество строк в test.txt:"
wc -l test.txt

# Создание файлов date.txt в каталогах, кратных 5
for i in {1..40}; do
  if [ $((i % 5)) -eq 0 ]; then
    echo $(date +"%Y-%m-%d") > "test$i/date.txt"
  fi
done

# Добавление времени в каталогах, кратных 10
for i in {1..40}; do
  if [ $((i % 10)) -eq 0 ]; then
    echo $(date +"%H:%M:%S") >> "test$i/date.txt"
  fi
done

echo "Lab is done!"
```

### Создайте файл `Dockerfile`:

```dockerfile
# Используем минимальный образ Ubuntu
FROM ubuntu:latest

# Устанавливаем рабочую директорию
WORKDIR /home

# Обновляем список пакетов и устанавливаем необходимые утилиты
RUN apt-get update && apt-get install -y bash wget curl

# Копируем скрипт в контейнер
COPY script.sh /usr/local/bin/script.sh

# Делаем скрипт исполняемым
RUN chmod +x /usr/local/bin/script.sh

# Выполняем скрипт при запуске контейнера
CMD ["/bin/bash", "/usr/local/bin/script.sh"]
```

### Сборка и запуск образа:

```bash
docker build -t lab105 .
docker run --rm lab105
```

- `docker build`: создает образ Docker.
- `-t lab105`: присваивает имя образу.
- `.`: использует Dockerfile из текущей директории.
- `docker run --rm`: запускает контейнер и удаляет его после завершения.
